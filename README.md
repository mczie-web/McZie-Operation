# McZie Operation

Base operativa para construir, ejecutar y escalar sistemas de agentes con estándares profesionales.

## Objetivo
Crear una plataforma de ejecución confiable para operaciones, ventas y administración asistidas por agentes.

## Estructura
- `docs/` documentación ejecutiva y técnica
- `src/` código fuente
- `tests/` pruebas
- `.github/workflows/` automatizaciones CI

## Primeros pasos
1. Clonar el repositorio.
2. Revisar `docs/ROADMAP.md`.
3. Ejecutar CI en cada PR.

## Estándar de trabajo
- Commits pequeños y claros.
- Pull Requests con contexto de negocio.
- Cambios con impacto medible y trazable.
